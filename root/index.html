このサイトついて

概要

このサイトはふらふらと当てもなくコンピュータ・ロボットのお勉強をしている筆者の備忘録です。 インターネットからかき集めた情報・実行して得られたコツなどを将来振り返るために記録しています。 これらの情報がこれから学習する方々の何らかの参考になればとてもうれしく思います。

このサイトの方向性

このサイトでは、Linux・組み込み系・ロボット・画像処理などをテーマに扱っていきます。 徐々にコンテンツを増やしていきますので、必要なものなどご活用ください。

筆者について

電気電子情報系の学生ですが、基本的には情報系しかできません。 パソコンの上だけで動くプログラムから、実際に動かすロボットのプログラムまで、興味のあるものは何でもつくります。 普段はArchLinux と Python を使っていろいろやってます。 熱しやすく冷めやすいので、いろいろ長続きするとは限りません。

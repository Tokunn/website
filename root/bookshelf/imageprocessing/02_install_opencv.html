OpenCVインストール

目的

LinuxにOpenCVのインストールをします。

準備

まずは下記のものを用意してください

  1. Linuxマシン

Ubuntu

Ubuntuの場合は以下のコマンドを実行してください。

                    $ sudo apt-get install libcv2.4 libhighgui2.4 libcvaux2.4 libcv-dev libcvaux-dev python-opencv opencv-doc

ArchLinux

ArchLinuxの場合は以下のコマンドを実行してください。

                    $ sudo pacman -Sy opencv

おわりに

これでインストールは完了です。
